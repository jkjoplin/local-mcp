Setup Wizard

Get MLX models running locally in minutes

Check Prerequisites

Verify Python and MLX are available on your system.

Install MLX

Install the MLX language model framework.

pip install mlx-lm

Download a Model

Pick a model from the library and download it.

huggingface-cli download mlx-community/Qwen2.5-1.5B-Instruct-4bit

Start MLX Server

Launch the model as an OpenAI-compatible server.

python3 -m mlx_lm.server --model mlx-community/Qwen2.5-1.5B-Instruct-4bit --port 8083

Configure local-mcp

Set environment variables or edit ~/.local-mcp/config.json.

export LOCAL_MCP_FAST_URL=http://localhost:8083 export LOCAL_MCP_SMART_URL=http://localhost:8081

Add to Claude Code

Register local-mcp as an MCP server.

claude mcp add local-mcp -- npx local-mcp serve
